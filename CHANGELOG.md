# Changelog

All notable changes to Finance Mentor AI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-10

### Added

#### Core Features
- Dashboard with real-time financial overview
- AI-powered spending forecast with 7D/30D/3M views
- Financial Pulse score with radial indicator
- Transaction management with search and filtering
- Budget creation and tracking
- Savings goals with progress tracking
- Analytics page with spending trends and comparisons

#### Financial Tools
- Investment Portfolio tracker (Stocks, Crypto, ETFs, Bonds)
- Net Worth Tracker (Assets vs Liabilities)
- Debt Tracker with payoff strategies
- Bill Reminders with recurring support
- Recurring Transactions management
- Advanced transaction filters

#### Calculators & Planners
- Tax Planner with Indian tax system (FY 2025-26)
  - Old vs New regime comparison
  - Section-wise deductions (80C, 80D, 80CCD, HRA)
  - Real-time tax calculation
- Credit Score Monitor (300-900 range)
  - Score factors breakdown
  - Credit account tracking
  - Improvement tips
- EMI Calculator
  - Multiple loan types (Home, Car, Personal, Education)
  - Interactive sliders
  - Amortization schedule

#### UI/UX Features
- Glassmorphism design with translucent panels
- Blue gradient theme with floating 3D shapes
- Smooth animations and transitions
- Responsive design for all screen sizes
- Dark mode optimized
- AI Chat Assistant with contextual responses
- Notification Center with real-time updates
- Quick Actions floating menu
- Spending Insights dashboard

#### Data Management
- Export transactions to CSV/JSON
- Import data from JSON
- Local storage persistence
- Data validation and error handling

#### Navigation
- Desktop header with all pages
- Mobile bottom navigation
- Tools dropdown menu
- Active page highlighting

### Technical
- React 18.3.1 with TypeScript
- Vite for fast development and building
- React Router for navigation
- Recharts for data visualization
- Tailwind CSS for styling
- Context API for state management
- Local storage for data persistence
